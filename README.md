# Prompt Engineering Assistant

## Description

A Python application that helps users generate, improve, explain, optimize, evaluate, and translate prompts using Google's Gemini AI.

## Features

- Generate Prompt
- Improve Prompt
- Explain Prompt
- Optimize Prompt
- Evaluate Prompt
- Translate Prompt
- AI Chat
- Save Prompt
- View History
- Export History

## Requirements

- Python 3.10+
- google-genai
- python-dotenv
- colorama

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
py -3.10 app.py
```
